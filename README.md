# :sparkles: MY DOTFILES :sparkles:

Configurações pessoais para Zsh, Bash e WSL.

## Instalação
```bash
git clone https://github.com/seu-usuario/dotfiles.git ~/dotfiles
cd ~/my-dotfiles
chmod +x install.sh
./install.sh
```

## Conteúdo

- **zsh/**: Configurações do Zsh
- **bash/**: Configurações do Bash
- **git/**: configurações do Git
